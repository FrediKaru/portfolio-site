## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone git@github.com:FrediKaru/portfolio-site.git
   ```
2. Locate repository:
   ```bash
   cd cloned-project
     ```
3. Install the dependencies:
   ```bash
   npm install
     ```
4. Install the dependencies:
   ```bash
   npm run dev
     ```
